<p align="center">
  <strong>Lapse</strong> is a lightweight Android utility designed to help you track exactly when applications on your device were installed and last updated.
</p>

<p align="center">
  Built with modern <strong>Material You</strong> design principles, Lapse focuses on simplicity, speed, and usability while remaining completely offline.
</p>

---

## ✨ Features

* **Clean Interface:** A fully searchable, clutter-free list of all your installed applications.
* **Dual Timestamps:** Displays precise dates alongside human-readable relative timestamps (e.g., *"Installed 2 days ago"*, *"Updated 4 weeks ago"*).
* **At-a-Glance Insights:** Easily identify newly installed apps, recent updates, or abandoned legacy software.
* **Material You:** Dynamically adapts to your device's theme and accent colors.

## 🔒 Privacy & Security

Lapse is engineered with a strict **privacy-first** architecture:

* 🚫 **No Internet Permission:** The app has no network access capabilities and cannot connect to the web.
* 🙅‍♂️ **No Data Collection:** Zero logs, analytics, tracking scripts, or telemetry.
* 📦 **Fully Offline:** All metadata processing happens locally on your device.
* 👤 **No Accounts Required:** No sign-ins, cloud dependencies, or online synchronization.

### Permissions
Lapse requests only the **absolute minimal permissions** required to fetch local application metadata from the Android OS.

## 🚀 Installation

1. Navigate to the **[Releases](../../releases)** section of this repository and download the latest APK.
2. If prompted, enable **"Install unknown apps"** for your browser or file manager.
3. Install the APK file.
4. Launch **Lapse** to instantly view your application insights.

### Compatibility
* **OS Version:** Android 10.0 (API level 29) and newer.
